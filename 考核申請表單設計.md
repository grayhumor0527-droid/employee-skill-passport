```
歡迎使用摩研裝修考核申請系統。

請在符合升級條件時填寫此表單，主管將在 3 個工作天內回覆審核結果。

【申請條件】
- 技能分數達到下一職級門檻
```

## 精簡版優點

| 項目 | 原版 | 精簡版 |
|------|:----:|:------:|
| 題數 | 17 題 | 9 題 |
| 簡答題 | 5 題 | 1 題 |
| 填寫時間 | 約 5 分鐘 | 約 1-2 分鐘 |
| 手機友善度 | 普通 | 高 |

### 設計理念

1. **姓名用下拉選單** - 避免打錯字，直接選擇
2. **手機末4碼驗證** - 簡單確認身分，取代輸入完整手機
3. **技能分數用區間** - 員工不一定記得精確分數
4. **案場用核取方塊** - 勾選比打字快
5. **移除自評分數** - 主管評分即可，避免重複

## Apps Script 自動化

### 設定步驟

1. 開啟 Google Forms 編輯頁面
2. 右上角 ⋮ → 指令碼編輯器
3. 貼上以下程式碼
4. 儲存
5. 觸發條件 → 新增 → onFormSubmit → 表單提交時

### 表單提交觸發器

```javascript
/**
 * 考核申請表單提交處理
 * 表單連結：https://forms.gle/nTHruXr1Yfc64CJG7
 */
function onFormSubmit(e) {
  const responses = e.namedValues;

  const empName = responses['姓名'][0];
  const phone4 = responses['手機末4碼（驗證用）'][0];
  const appType = responses['申請類型'][0];
  const currentRank = responses['目前職級'][0];
  const scoreRange = responses['技能分數區間'][0];
  const projects = responses['主要參與案場'] ? responses['主要參與案場'].join('、') : '';
  const skills = responses['技術專長'] ? responses['技術專長'].join('、') : '';
  const reason = responses['申請原因'][0];
  const note = responses['其他說明（選填）'] ? responses['其他說明（選填）'][0] : '';

  const appId = 'APP-' + Utilities.formatDate(new Date(), 'Asia/Taipei', 'yyyyMMdd-HHmmss');

  // 發送 Email 通知主管
  const subject = '【摩研】考核申請 - ' + empName;
  const body = `
考核申請通知

申請編號：${appId}
員工姓名：${empName}
申請類型：${appType}
目前職級：${currentRank}
技能分數：${scoreRange}
代表案場：${projects}
技術專長：${skills}

申請原因：
${reason}

${note ? '其他說明：\n' + note : ''}

---
請至 Google Sheets 審核此申請
  `;

  MailApp.sendEmail({
    to: 'your-email@gmail.com',  // 改成主管 Email
    subject: subject,
    body: body
  });

  // 記錄到試算表（可選）
  // SpreadsheetApp.openById('YOUR_SHEET_ID').getSheetByName('考核申請').appendRow([...]);
}
```

## 考核評分表（主管用）

### 評分項目

| 項目 | 權重 | 1分 | 2分 | 3分 | 4分 | 5分 |
|------|:----:|-----|-----|-----|-----|-----|
| 技術能力 | 40% | 欠缺 | 不足 | 合格 | 純熟 | 卓越 |
| 工作態度 | 20% | 不佳 | 被動 | 正常 | 負責 | 主動 |
| 出勤狀況 | 15% | <80% | 80-89% | 90-94% | 95-97% | ≥98% |
| 團隊合作 | 15% | 困難 | 摩擦 | 正常 | 良好 | 核心 |
| 學習成長 | 10% | 停滯 | 緩慢 | 正常 | 積極 | 主動 |

### 評分計算

```
總分 = 技術×0.4 + 態度×0.2 + 出勤×0.15 + 團隊×0.15 + 學習×0.1
```

### 等第判定

| 總分 | 等第 | 後續處理 |
|-----:|:----:|----------|
| 4.5-5.0 | 優等 | 優先升級、獎金加成 |
| 4.0-4.4 | 甲等 | 正常升級 |
| 3.0-3.9 | 乙等 | 暫緩升級 |
| 2.0-2.9 | 丙等 | 列入觀察 |
| <2.0 | 丁等 | 輔導改善 |
